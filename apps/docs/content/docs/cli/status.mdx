# morpheus status

Check the status of your deployment, including resource usage and escrow balance.

## Usage

```bash
morpheus status [options] [dseq]
```

## Arguments

| Argument | Description                                                         |
| -------- | ------------------------------------------------------------------- |
| `dseq`   | Deployment sequence number (optional, uses latest if not specified) |

## Options

| Option        | Description                            |
| ------------- | -------------------------------------- |
| `--json`      | Output as JSON                         |
| `--watch, -w` | Continuously update status             |
| `--interval`  | Watch interval in seconds (default: 5) |

```bash
  Rate: 0.0015 AKT/block

  ✓ postgres  Running    (healthy)
  ✓ vector    Running    (healthy)

Uptime: 5 days, 3 hours
```

## Status Values

| Status      | Description                    |
| ----------- | ------------------------------ |
| `Active`    | Deployment is running normally |
| `Deploying` | Deployment is starting up      |
| `Stopped`   | Deployment is stopped          |
| `Failed`    | Deployment failed to start     |
| `Closing`   | Deployment is being closed     |

## Watch Mode

Continuously monitor status:

```bash
morpheus status --watch

# Update every 10 seconds
morpheus status --watch --interval 10
```

Press `Ctrl+C` to stop watching.

## List All Deployments

See all your deployments:

```bash
morpheus status --all

DSEQ        Name            Status    URL                         Created
12345678    my-agent        Active    abc123.akash.network        5d ago
12345679    mcp-server      Active    def456.akash.network        3d ago
12345680    landing-page    Stopped   ghi789.akash.network        1w ago
```

## Examples

Check latest deployment:

```bash
morpheus status
```

Check specific deployment:

```bash
morpheus status 12345678
```

Monitor in real-time:

```bash
morpheus status --watch
```

Get JSON for scripting:

```bash
morpheus status --json | jq '.escrow.estimatedDaysRemaining'
```
