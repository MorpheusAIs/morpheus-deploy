# morpheus deploy

Deploy your application to Akash Network. This command builds your container, synthesizes SDL, and creates a deployment.

## Usage

```bash
morpheus deploy [options]
```

## Options

| Option         | Description                                       |
| -------------- | ------------------------------------------------- |
| `--dry-run`    | Preview deployment without executing              |
| `--skip-build` | Use existing image, skip Docker build             |
| `--skip-push`  | Skip pushing to registry (use with --skip-build)  |
| `--image`      | Specify a custom image to deploy                  |
| `--provider`   | Preferred provider address                        |
| `--wait`       | Wait for deployment to be healthy (default: true) |
| `--timeout`    | Deployment timeout in seconds (default: 300)      |

## Dry Run

Preview what would happen without deploying:

```bash
morpheus deploy --dry-run
```

This outputs:

- Validated configuration
- Generated SDL
- Estimated costs
- No actual deployment

## Using Custom Images

Deploy a pre-built image:

```bash
morpheus deploy --image ghcr.io/user/my-agent:v1.2.3
```

Skip both build and push:

```bash
morpheus deploy --skip-build --skip-push --image my-registry/my-image:tag
```

## Provider Selection

Prefer a specific provider:

```bash
morpheus deploy --provider akash1abc123...
```

## Timeouts

Adjust the deployment timeout:

```bash
morpheus deploy --timeout 600  # 10 minutes
```

## Examples

Standard deployment:

```bash
morpheus deploy
```

Quick redeploy with existing image:

```bash
morpheus deploy --skip-build
```

Preview before deploying:

```bash
morpheus deploy --dry-run
```

Deploy to testnet:

```bash
morpheus deploy --network testnet
```
