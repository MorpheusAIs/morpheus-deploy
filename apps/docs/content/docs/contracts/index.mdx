# Blockchain Integration

Morpheus operates across multiple blockchain networks to enable decentralized, crypto-native deployments. This section covers the blockchain components that power the platform.

## Multi-Chain Architecture

```
┌─────────────────────────────────────────────────────────────────┐
│                      Base Network (EVM)                          │
│  ┌─────────────────┐  ┌─────────────────┐  ┌─────────────────┐  │
│  │  Smart Wallet   │  │   MOR Staking   │  │      USDC       │  │
│  │   (ERC-4337)    │  │    Contract     │  │    Payments     │  │
│  └────────┬────────┘  └────────┬────────┘  └────────┬────────┘  │
└───────────│─────────────────────│─────────────────────│──────────┘
            │                     │                     │
            │                     │                     │
            └─────────────────────┼─────────────────────┘
                                  │
                           ┌──────▼──────┐
                           │  Skip Go    │
                           │ Cross-Chain │
                           │    Swap     │
                           └──────┬──────┘
                                  │
┌─────────────────────────────────│────────────────────────────────┐
│                      Akash Network (Cosmos)                      │
│  ┌─────────────────┐  ┌────────▼────────┐  ┌─────────────────┐  │
│  │  AuthZ Grants   │  │   Deployments   │  │    Escrow       │  │
│  │ (Permissions)   │  │   & Leases      │  │   Accounts      │  │
│  └─────────────────┘  └─────────────────┘  └─────────────────┘  │
└──────────────────────────────────────────────────────────────────┘
```

## Components

### Base Network (Mainnet & Sepolia Testnet)

| Component                                    | Purpose                      | Network |
| -------------------------------------------- | ---------------------------- | ------- |
| [Smart Wallet](/docs/contracts/smart-wallet) | ERC-4337 account abstraction | Base    |
| [MOR Staking](/docs/contracts/staking)       | Stake MOR for yield          | Base    |
| USDC                                         | Payment currency             | Base    |

### Akash Network (Mainnet & Sandbox Testnet)

| Component                                   | Purpose                 | Network |
| ------------------------------------------- | ----------------------- | ------- |
| [AuthZ Grants](/docs/contracts/authz)       | Permission delegation   | Akash   |
| [Deployments](/docs/contracts/akash-client) | Container orchestration | Akash   |
| Escrow                                      | Lease payments          | Akash   |

### Cross-Chain Bridge

USDC on Base is swapped to AKT on Akash via Skip Go Protocol:

```
USDC (Base) → Skip Go API → AKT (Akash)
```

## Usage

```typescript
import {
  SmartWalletManager,
  EphemeralKeyManager,
  AuthZManager,
  MorpheusStaking,
  AkashClient,
} from '@morpheus-deploy/contracts';

// Create smart wallet
const wallet = new SmartWalletManager({ network: 'testnet' });
const walletInfo = await wallet.createSmartWallet();

// Generate ephemeral key
const ephemeral = new EphemeralKeyManager();
const key = await ephemeral.generate(['deploy', 'status']);

// Create AuthZ grant
const authz = new AuthZManager();
await authz.createDeploymentGrant(granterAddress, key.address);

// Deploy to Akash
const akash = new AkashClient({ network: 'testnet' });
await akash.createDeployment({ owner, sdl, deposit });
```

## Security Model

Morpheus uses a layered security approach:

1. **Smart Wallet** - Passkey-based authentication, no seed phrases
2. **Ephemeral Keys** - Time-limited keys that auto-expire
3. **AuthZ Grants** - Minimal permissions, limited to deployment operations
4. **Sealed Secrets** - ECIES encryption for environment variables

See each component's documentation for detailed security information.
